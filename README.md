# D2C Unit Economics — SQL Project

## Objective
Analyze key unit economics metrics for Indian D2C brands using MySQL.

## Metrics Covered
| Metric | File |
|---|---|
| CAC by channel | 03_cac_analysis.sql |
| LTV & LTV:CAC | 04_ltv_analysis.sql |
| Contribution Margin | 05_contribution_margin.sql |
| Cohort & Repeat Rate | 06_cohort_repeat_rate.sql |
| Payback Period | 07_payback_period.sql |

## Key Findings (sample data)
- LTV:CAC ratio benchmarked across 5 D2C brands
- Cohort retention tracked over 3 months
- Payback period flagged as healthy / borderline / critical

## Tools
- MySQL 8.0+
- Sample data inspired by Indian D2C ecosystem
